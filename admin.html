GitHub Erişimi

Bu panel değişiklikleri doğrudan github.com/martivstaxi/TR81 deposuna işler. Cloudflare Pages otomatik deploy eder → canlı sitede birkaç dakikada güncellenir.

Token nasıl alınır: GitHub → Settings → Developer settings → Fine-grained tokens → Generate → Repository access: Only select repositories → TR81 → Permissions → Repository permissions → Contents: Read and write. Oluştur, kopyala, buraya yapıştır.
Token yalnızca senin tarayıcında saklanır (localStorage), hiçbir yere gönderilmez.
